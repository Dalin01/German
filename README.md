# 🇩🇪 German B1/B2 — 12-Week Study Plan

Daily study notes for German language learning, progressing from B1 consolidation through B2 mastery.

## Setup

- **Level**: B1 → B2 (mixed, progressive)
- **Duration**: 12 weeks (60 study days)
- **Schedule**: Monday–Friday, 60 min/day
- **Start date**: 2026-04-28

## Daily Session Structure

| Block | Duration | Focus |
|-------|----------|-------|
| Grammar | 20 min | Rules, examples, exercises |
| Vocabulary | 15 min | 15–20 words by topic |
| Reading | 15 min | B1/B2 text + comprehension |
| Writing | 10 min | Prompted writing task |

## Curriculum Phases

1. **Weeks 1–4** — B1 Consolidation & B2 Bridge (everyday topics, core grammar review)
2. **Weeks 5–8** — Core B2 (abstract topics, complex grammar, opinion texts)
3. **Weeks 9–12** — Advanced B2 & Integration (exam prep, essay writing, literary texts)
