## Lektion 10: Karriere — „Erfolgreich scheitern"

### Grammatik

#### 1. Nomen-Verb-Verbindungen (Fortsetzung)

| Nomen-Verb-Verbindung | Einfaches Verb | Beispiel |
|------------------------|---------------|----------|
| einen Fehler **machen** | sich irren | *Jeder **macht** mal einen **Fehler**.* |
| Erfahrungen **sammeln** | erleben, erfahren | *Sie hat im Ausland viele **Erfahrungen gesammelt**.* |
| eine Lösung **finden** | lösen | *Wir müssen eine **Lösung finden**.* |
| Feedback **geben** | rückmelden | *Der Coach **gibt** ihr konstruktives **Feedback**.* |
| in Anspruch **nehmen** | nutzen | *Er **nimmt** ein Coaching in **Anspruch**.* |
| zur Sprache **bringen** | ansprechen | *Sie **bringt** das Problem zur **Sprache**.* |
| einen Vortrag **halten** | vortragen | *Er **hält** einen **Vortrag** über Karriereplanung.* |
| sich Mühe **geben** | sich anstrengen | *Er **gibt** sich große **Mühe**.* |

#### 2. Temporale Zusammenhänge: Zeitbezüge herstellen und Zeitangaben machen

| Ausdruck | Zeitbezug | Beispiel |
|----------|-----------|----------|
| bevor / ehe | Vorzeitigkeit (vorher) | ***Bevor** er Coach wurde, war er Manager.* |
| nachdem | Nachzeitigkeit (danach) | ***Nachdem** er gescheitert war, gründete er eine eigene Firma.* |
| während | Gleichzeitigkeit | ***Während** er arbeitete, studierte er nebenbei.* |
| seitdem / seit | ab einem Zeitpunkt bis jetzt | ***Seitdem** er selbstständig ist, ist er zufriedener.* |
| bis | bis zu einem Zeitpunkt | *Er arbeitete dort, **bis** die Firma schloss.* |
| sobald | unmittelbar danach | ***Sobald** er den Job verlor, begann er mit dem Coaching.* |
| solange | für die gesamte Dauer | ***Solange** er motiviert ist, wird er Erfolg haben.* |

> **Tempus bei *nachdem*:**
> Plusquamperfekt → Präteritum: ***Nachdem** er die Firma verloren **hatte**, **gründete** er eine neue.*
> Perfekt → Präsens: ***Nachdem** er die Ausbildung **abgeschlossen hat**, **arbeitet** er als Coach.*

### Wortbildung: Nomen aus Adjektiven (Partizip Präsens und Partizip Perfekt als Nomen)

| Partizip | Als Nomen | Beispiel |
|----------|-----------|----------|
| der/die Angestellte | (Partizip II) | *Die **Angestellten** sind zufrieden.* |
| der/die Vorgesetzte | (Partizip II) | *Mein **Vorgesetzter** gibt mir Feedback.* |
| der/die Arbeitsuchende | (Partizip I) | *Die **Arbeitsuchenden** bewerben sich.* |
| der/die Auszubildende | (Partizip I) | *Der **Auszubildende** lernt den Beruf.* |
| der/die Selbstständige | (Adj.) | *Als **Selbstständiger** ist man flexibel.* |
| der/die Berufstätige | (Adj.) | *Viele **Berufstätige** klagen über Stress.* |

> **Deklination:** These nouns decline like adjectives!
> *ein Angestellt**er*** / *der Angestellt**e*** / *die Angestellt**en***

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Argumente einschränken | *Das stimmt zwar, aber …* / *Das gilt nur bedingt.* / *Das mag in manchen Fällen so sein, aber …* |
| Über Stärken sprechen | *Meine größte Stärke ist, dass …* / *Ich bin gut darin, …* / *Besonders gut kann ich …* |
| Über Schwächen sprechen | *Manchmal fällt es mir schwer, …* / *Daran muss ich noch arbeiten.* / *Eine Schwäche von mir ist …* |
| Über Fehler reflektieren | *Im Nachhinein hätte ich …* / *Daraus habe ich gelernt, dass …* / *Dieser Fehler hat mir gezeigt, dass …* |

## Lektion 11: Konsum — „Weniger ist mehr!"

### Grammatik

#### 1. Synonyme: Allgemeinsprache vs. Umgangssprache

| Allgemeinsprache | Umgangssprache | Bedeutung |
|-----------------|---------------|-----------|
| wegwerfen | schmeißen | to throw away |
| kaputtgehen | den Geist aufgeben | to break down |
| kaufen | sich zulegen | to buy, get |
| sparsam | knauserig | thrifty / stingy |
| Unsinn | Quatsch, Blödsinn | nonsense |
| sich ärgern | sich aufregen | to get annoyed |
| sehr | echt, total, mega | very |

> **B2 Kompetenz:** You need to recognize both registers and choose the appropriate one for context (formal writing vs. casual speaking).

#### 2. Modale Zusammenhänge: Mittel und Umstände beschreiben

| Ausdruck | Bedeutung | Beispiel |
|----------|-----------|----------|
| indem | by doing (method) | *Man spart Geld, **indem** man weniger kauft.* |
| dadurch, dass | by the fact that | ***Dadurch, dass** man Dinge repariert, reduziert man Müll.* |
| durch (+Akk) | through (means) | ***Durch** bewussten Konsum kann man die Umwelt schützen.* |
| mithilfe von / mit (+Dat) | with the help of | ***Mithilfe von** Apps kann man seinen Konsum analysieren.* |
| auf diese Weise / so | in this way | ***Auf diese Weise** kann man viel Geld sparen.* |
| ohne … zu | without doing | *Sie lebt minimalistisch, **ohne** dabei auf Komfort **zu verzichten**.* |
| anstatt … zu | instead of doing | ***Anstatt** neue Sachen **zu kaufen**, repariert sie alte.* |

> **Achtung:** *indem* = **wie** man etwas macht (Methode). *dadurch, dass* = betont das **Ergebnis**.
> *Indem man weniger kauft, spart man Geld.* (Fokus: Methode)
> *Dadurch, dass man weniger kauft, spart man Geld.* (Fokus: Ergebnis)

### Wortbildung: Verben mit den Präfixen *ver-*, *ent-* und *weg-*

| Präfix | Verb | Bedeutung | Beispiel |
|--------|------|-----------|----------|
| **ver-** | verbrauchen | to consume, use up | *Wir verbrauchen zu viel Plastik.* |
| **ver-** | verzichten (auf) | to do without | *Ich verzichte auf Plastiktüten.* |
| **ver-** | verschwenden | to waste | *Verschwende kein Essen!* |
| **ent-** | entrümpeln | to declutter | *Am Wochenende entrümple ich meine Wohnung.* |
| **ent-** | entsorgen | to dispose of | *Bitte entsorgen Sie den Müll richtig.* |
| **ent-** | sich entscheiden | to decide | *Ich habe mich für Minimalismus entschieden.* |
| **weg-** | wegwerfen | to throw away | *Wirf es nicht einfach weg!* |
| **weg-** | weggeben | to give away | *Ich gebe alte Kleidung weg.* |

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Einschätzungen formulieren | *Ich halte das für …* / *Meiner Einschätzung nach …* / *So wie ich das sehe, …* |
| Verzicht ausdrücken | *Ich kann gut auf … verzichten.* / *Ich brauche eigentlich kein/e …* / *Darauf kann ich gut verzichten.* |
| Argumente einschränken | *Das ist zwar richtig, aber …* / *Das gilt nur eingeschränkt.* / *Man muss aber auch bedenken, dass …* |

## Lektion 12: Zusammen Leben — „Auf gute Nachbarschaft!"

### Grammatik

#### 1. Passiversatzformen
Alternatives to passive constructions — very common in written B2 German:

| Passiv | Ersatzform | Beispiel |
|--------|-----------|----------|
| kann … werden | sich lassen + Infinitiv | *Das **lässt sich** leicht lösen.* |
| kann … werden | sein + zu + Infinitiv | *Das **ist** leicht **zu lösen**.* |
| kann … werden | -bar / -lich Adjektiv | *Das ist leicht **lösbar**.* |
| muss … werden | sein + zu + Infinitiv | *Der Antrag **ist** bis Freitag **abzugeben**.* (= muss abgegeben werden) |
| kann nicht … werden | un- + -bar / -lich | *Das Problem ist **unlösbar**.* |

**Full comparison:**
| Passiv | sich lassen | sein + zu | Adjektiv auf -bar/-lich |
|--------|-----------|-----------|------------------------|
| *Das kann gemacht werden.* | *Das **lässt sich** machen.* | *Das **ist** zu machen.* | *Das ist **machbar**.* |
| *Das kann nicht erklärt werden.* | *Das **lässt sich** nicht erklären.* | *Das **ist** nicht zu erklären.* | *Das ist **unerklärlich**.* |

#### 2. Mehrdeutige Wörter
Words with multiple meanings depending on context:

| Wort | Bedeutung 1 | Bedeutung 2 |
|------|-------------|-------------|
| die Bank | bank (financial) | bench |
| der Schloss | castle | lock |
| der Absatz | paragraph | heel (shoe) / sales |
| die Leiter | ladder | leader (der Leiter) |
| der Zug | train | trait, pull, draft |
| der Strauß | bouquet | ostrich |
| die Birne | pear | lightbulb |

### Wortbildung: Adjektive mit *-bar*, *-lich* und *-fähig*

| Suffix | Bedeutung | Beispiel |
|--------|-----------|----------|
| **-bar** | can be done | mach**bar** (doable), trink**bar** (drinkable), erreich**bar** (reachable) |
| **-lich** | can be done / relating to | verständ**lich** (understandable), freund**lich** (friendly), nach**lich** |
| **-fähig** | capable of / able to | leistungs**fähig** (capable), wettbewerbs**fähig** (competitive), team**fähig** (team player) |

### Wortbildung: Fugenelement *-s-* nach bestimmten Suffixen

After these suffixes, compound nouns use a linking **-s-**:

| Suffix | Beispiel |
|--------|----------|
| **-heit** | Gesund**heit**s**-**amt, Frei**heit**s**-**kämpfer |
| **-(at)ion** | Kommunika**tion**s**-**problem, Organisa**tion**s**-**talent |
| **-ität** | Qual**ität**s**-**kontrolle, Aktiv**ität**s**-**level |
| **-keit** | Mög**lichkeit**s**-**form, Schwierig**keit**s**-**grad |
| **-schaft** | Nachbar**schaft**s**-**fest, Freund**schaft**s**-**dienst |
| **-ung** | Wohn**ung**s**-**suche, Bewer**bung**s**-**gespräch |

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Verärgerung ausdrücken | *Es stört mich, dass …* / *Es nervt mich, wenn …* / *Ich finde es unmöglich, dass …* / *Das geht einfach nicht!* |
| Sich rechtfertigen | *Das war nicht meine Absicht.* / *Ich habe doch nur …* / *Das liegt daran, dass …* / *Lassen Sie mich erklären, …* |
| Sich einigen | *Können wir uns darauf einigen, dass …?* / *Wie wäre es, wenn …?* / *Einverstanden, dann machen wir das so.* |
| Ein Projekt vorstellen | *Ich möchte Ihnen unser Projekt vorstellen.* / *Unser Ziel ist es, …* / *Das Besondere an diesem Projekt ist, dass …* |

## Aussprache-Fokus Modul 4
1. **Die Konsonantenverbindung *ts*** — Appears as *z* (*Zeit*), *tz* (*Platz*), *ts* (*nichts*), *-tion* (*Kommunika**tion***) — all pronounced [ts]
2. **Der Vokal *ü*** — Lips rounded like *u*, tongue positioned like *i*: *über, Tür, Stück, Glück, Müll*
3. **Mit Emotion sprechen** — Practice expressing anger, surprise, and enthusiasm with appropriate intonation
