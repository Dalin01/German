## 📖 Grammar (20 min) — Sätze verbinden (Combining Sentences)

### The Goal

Transform choppy, simple sentences into **smooth, complex B2-level writing**. This is the #1 skill that separates B1 from B2 writing.

### B1 Writing (choppy):
> Die Forscherin machte eine Entdeckung. Die Entdeckung war wichtig. Sie veröffentlichte die Ergebnisse. Die Ergebnisse erschienen in einer Fachzeitschrift.

### B2 Writing (combined):
> Die Forscherin, die eine wichtige Entdeckung gemacht hatte, veröffentlichte ihre Ergebnisse in einer renommierten Fachzeitschrift.

### Combining Techniques

#### 1. Relative Clauses (add detail)

**Separate:** Der Student bestand die Prüfung. Er hatte wochenlang gelernt.
**Combined:** Der Student, **der wochenlang gelernt hatte**, bestand die Prüfung.

#### 2. Participial Phrases (concise detail)

**Separate:** Die Methode wurde von Professor Schmidt entwickelt. Sie wird jetzt weltweit eingesetzt.
**Combined:** Die **von Professor Schmidt entwickelte** Methode wird jetzt weltweit eingesetzt.

#### 3. Nachdem / Bevor / Während (temporal)

**Separate:** Er beendete sein Studium. Danach begann er mit der Promotion.
**Combined:** **Nachdem** er sein Studium beendet **hatte**, begann er mit der Promotion.

#### 4. Indem / Dadurch, dass (method)

**Separate:** Er verbesserte sein Deutsch. Er las jeden Tag Zeitung.
**Combined:** Er verbesserte sein Deutsch, **indem** er jeden Tag Zeitung las.

#### 5. Obwohl / Trotzdem (concession)

**Separate:** Die Finanzierung war gering. Das Projekt war trotzdem erfolgreich.
**Combined:** **Obwohl** die Finanzierung gering war, war das Projekt erfolgreich.

#### 6. So... dass (result)

**Separate:** Die Rede war beeindruckend. Alle waren überzeugt.
**Combined:** Die Rede war **so** beeindruckend, **dass** alle überzeugt waren.

#### 7. um... zu / damit (purpose)

**Separate:** Sie lernte Deutsch. Sie wollte in Deutschland studieren.
**Combined:** Sie lernte Deutsch, **um** in Deutschland **zu** studieren.

#### 8. Apposition (add identity)

**Separate:** Berlin ist die Hauptstadt. Berlin hat viele Universitäten.
**Combined:** **Berlin, die Hauptstadt Deutschlands,** hat viele Universitäten.

### Combination Strategies for Exam Writing

**Strategy 1: Lead with the background**
→ Nachdem... / Weil... / Da... → Hauptsatz

**Strategy 2: Embed detail in the middle**
→ Hauptsatz₁ + Relativsatz + Hauptsatz₁ (weiter)

**Strategy 3: Add consequence at the end**
→ Hauptsatz, so dass... / weshalb... / woraufhin...

## 📝 Wortschatz (15 min) — Redemittel: Textverknüpfung

| Deutsch | English | Function |
|---------|---------|----------|
| Wie bereits erwähnt, ... | As already mentioned, ... | Rückbezug |
| In diesem Zusammenhang ... | In this context ... | Thema vertiefen |
| Diesbezüglich ... | In this regard ... | Bezugnahme |
| Vor diesem Hintergrund ... | Against this background ... | Kontext geben |
| Damit zusammenhängend ... | Related to this ... | Verknüpfung |
| Was ... betrifft, ... | As far as ... is concerned, ... | Themenwechsel |
| Was ... angeht, ... | Regarding ... | Themenwechsel (informeller) |
| Der Grund dafür ist, dass ... | The reason for this is that ... | Erklärung |
| Dies hat zur Folge, dass ... | This has the consequence that ... | Konsequenz |
| Daran zeigt sich, dass ... | This shows that ... | Schlussfolgerung |
| Es kommt hinzu, dass ... | In addition, ... | Ergänzung |
| Nicht zuletzt ... | Not least ... | Letzter wichtiger Punkt |
| Im Vergleich zu + Dat. | Compared to ... | Vergleich |
| Im Unterschied zu + Dat. | In contrast to ... | Kontrast |
| Mit anderen Worten ... | In other words ... | Umformulierung |
| Das heißt (d.h.), ... | That is (i.e.), ... | Erklärung |
| Genauer gesagt ... | More precisely ... | Präzisierung |
| Kurzum ... | In short ... | Zusammenfassung |

## 📚 Lesen (15 min) — Text Cohesion Analysis

### Lesetext

**Die Bedeutung von Mehrsprachigkeit in der modernen Welt**

In einer globalisierten Welt spielen Fremdsprachen eine immer wichtigere Rolle. Wie bereits erwähnt in zahlreichen Studien, fördert Mehrsprachigkeit nicht nur die kognitive Entwicklung, sondern verbessert auch die beruflichen Chancen.

In diesem Zusammenhang ist es bemerkenswert, dass Deutschland, trotz seiner wirtschaftlichen Stärke, in der Fremdsprachenausbildung hinter einigen europäischen Nachbarländern zurückbleibt. Vor diesem Hintergrund haben mehrere Bundesländer begonnen, den Fremdsprachenunterricht bereits in der Grundschule einzuführen. Dies hat zur Folge, dass Kinder heute früher mit einer zweiten Sprache in Berührung kommen als noch vor zwanzig Jahren.

Was die Erwachsenenbildung betrifft, so bieten Volkshochschulen, die größten Anbieter von Sprachkursen in Deutschland, ein breites Programm an. Im Vergleich zu privaten Sprachschulen sind die Kurse dort deutlich günstiger. Nicht zuletzt tragen auch digitale Lernplattformen dazu bei, dass Sprachenlernen heute flexibler und zugänglicher ist.

Kurzum: Mehrsprachigkeit ist keine Luxuskompetenz mehr, sondern eine Notwendigkeit. Vor diesem Hintergrund müssen Bildungssysteme weltweit ihre Strategien anpassen, indem sie Sprachförderung als zentrales Element der Ausbildung etablieren.
