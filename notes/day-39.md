## 📖 Grammar (20 min) — Konditionalsätze (wenn...dann, falls, sofern)

### Real Conditionals (Reale Konditionalsätze)

Real conditionals describe **possible/likely** situations. They use the **Indikativ** (normal verb forms).

#### wenn — "if/when"

- **Wenn du sparst, kannst du dir ein Haus leisten.** (If you save, you can afford a house.)
- **Wenn die Zinsen steigen, wird der Kredit teurer.** (If interest rates rise, the loan gets more expensive.)

#### falls — "in case / if (uncertain)"

*Falls* implies the condition is **less likely** or **uncertain**:

- **Falls die Versicherung nicht zahlt, müssen wir selbst aufkommen.** (In case the insurance doesn't pay, we'll have to cover it ourselves.)
- **Falls Sie Fragen haben, rufen Sie uns an.** (If you have questions, call us.)

#### sofern — "provided that / as long as" (formal)

*Sofern* implies the condition **must be met**:

- **Sofern alle Unterlagen vorliegen, wird der Antrag bearbeitet.** (Provided that all documents are submitted, the application will be processed.)
- **Sofern Sie die Bedingungen erfüllen, erhalten Sie den Kredit.** (As long as you meet the conditions, you'll receive the loan.)

### Word Order

All three are subordinating conjunctions → **verb at the end** in the conditional clause:

- **Wenn** man den Vertrag kündigt, muss man eine Frist einhalten.
- Main clause first: Man muss eine Frist einhalten, **wenn** man den Vertrag kündigt.

### Useful patterns

- **Wenn... dann**: **Wenn** du investierst, **dann** kannst du Rendite erzielen.
- **Nur wenn**: **Nur wenn** Sie pünktlich überweisen, bleibt der Vertrag gültig.
- **Selbst wenn / Auch wenn**: **Selbst wenn** die Zinsen sinken, lohnt sich das Sparkonto nicht.

### Examples in Contract/Insurance Context

1. **Wenn Sie den Vertrag innerhalb von 14 Tagen widerrufen, bekommen Sie Ihr Geld zurück.**
2. **Falls ein Schaden eintritt, melden Sie ihn sofort der Versicherung.**
3. **Sofern die Kündigungsfrist eingehalten wird, endet der Vertrag zum Monatsende.**
4. **Selbst wenn die Versicherung teuer ist, lohnt sie sich langfristig.**
5. **Nur wenn alle Raten pünktlich gezahlt werden, bleibt der Kredit zinsfrei.**

## 📚 Lesen (15 min) — Insurance Comparison Text

### Lesetext

**Versicherungen in Deutschland — Was braucht man wirklich?**

Wenn man nach Deutschland zieht, wird man schnell mit einer Vielzahl von Versicherungen konfrontiert. Aber welche sind wirklich nötig?

**Pflichtversicherungen**

Manche Versicherungen sind gesetzlich vorgeschrieben. Wenn Sie in Deutschland leben, müssen Sie eine Krankenversicherung haben. Falls Sie ein Auto besitzen, ist eine Kfz-Haftpflichtversicherung Pflicht. Sozialversicherungsbeiträge (Rente, Arbeitslosenversicherung, Pflegeversicherung) werden automatisch vom Gehalt abgezogen.

**Empfohlene Versicherungen**

Die private Haftpflichtversicherung ist die wichtigste freiwillige Versicherung. Wenn Sie zum Beispiel versehentlich den Laptop eines Freundes beschädigen, können die Kosten schnell in die Tausende gehen. Eine Haftpflichtversicherung kostet nur etwa 50–80 Euro pro Jahr, deckt aber Schäden bis zu mehreren Millionen ab.

Falls Sie eine Wohnung mieten, ist eine Hausratversicherung sinnvoll. Sofern Ihre persönlichen Gegenstände durch Feuer, Einbruch oder Wasserschaden zerstört werden, erstattet die Versicherung den Wert.

Eine Berufsunfähigkeitsversicherung schützt Sie, falls Sie aus gesundheitlichen Gründen nicht mehr arbeiten können. Sie ist teurer, aber Experten empfehlen sie dringend.

**Vertragsdetails beachten**

Bevor Sie eine Versicherung abschließen, sollten Sie die Bedingungen genau lesen. Die typische Laufzeit beträgt ein Jahr mit automatischer Verlängerung. Wenn Sie kündigen möchten, müssen Sie die Kündigungsfrist von meist drei Monaten beachten. Falls Sie innerhalb von 14 Tagen nach Abschluss Bedenken haben, können Sie den Vertrag widerrufen.

**Tipp:** Vergleichsportale im Internet helfen, die günstigste Versicherung zu finden. Sofern Sie sich unsicher sind, bieten Verbraucherzentralen unabhängige Beratung an.

### Verständnisfragen

1. Welche Versicherungen sind in Deutschland Pflicht?
2. Was deckt eine private Haftpflichtversicherung ab?
3. Wie viel kostet eine Haftpflichtversicherung ungefähr pro Jahr?
4. Was passiert, wenn man die Kündigungsfrist nicht beachtet?
5. Wo kann man unabhängige Beratung zu Versicherungen bekommen?
