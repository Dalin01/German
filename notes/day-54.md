## 📖 Grammar (20 min) — Registerbewusstsein (Register Awareness)

### What is Register?

Register is the level of **formality** in language. At B2, you must be able to switch between formal and informal German consciously. The B2 exam specifically tests whether you can write in the **appropriate register**.

### The Three Main Registers

#### 1. Formell (Formal — essays, business letters, reports)

**Characteristics:**
- Sie-Form (Höflichkeitsform)
- Complex sentence structures (Nebensätze, Passiv, Konjunktiv)
- Nomen-Verb-Verbindungen instead of simple verbs
- No contractions, no slang
- Nominalization preferred

**Example:**
> Sehr geehrte Damen und Herren, hiermit möchte ich Ihnen mitteilen, dass die Ergebnisse der Untersuchung vorliegen. Es wäre wünschenswert, wenn Sie diese in Betracht ziehen würden.

#### 2. Standardsprachlich (Standard — neutral articles, presentations)

**Characteristics:**
- Can use Sie or man
- Mix of simple and complex structures
- Clear and direct but grammatically correct
- Some Redemittel used

**Example:**
> Die Studie zeigt, dass digitales Lernen effektiv ist. Allerdings sollte man auch die Nachteile berücksichtigen.

#### 3. Informell (Informal — emails to friends, personal texts)

**Characteristics:**
- du/ihr-Form
- Short sentences, colloquial expressions
- Contractions allowed (hab' ich, geht's)
- Modalpartikeln (doch, mal, ja, halt, eben)
- Emoticons/casual tone

**Example:**
> Hey, stell dir vor — hab die Prüfung doch tatsächlich bestanden! Hätte ich echt nicht gedacht, war ja mega schwer.

### Register Comparison Table

| Feature | Formell | Standard | Informell |
|---------|---------|----------|-----------|
| Anrede | Sehr geehrte/r... | (varies) | Hey / Hallo / Hi |
| Pronomen | Sie | man / Sie | du / ihr |
| Verb style | NVV, Passiv | Active + some Passiv | Simple verbs |
| Wortschatz | mitteilen, erörtern | sagen, besprechen | erzählen, quatschen |
| Satzlänge | Lang, verschachtelt | Mittel | Kurz |
| erhalten | empfangen/erhalten | bekommen | kriegen |
| beginnen | beginnen/einleiten | anfangen | loslegen |
| Negation | keineswegs | nicht | nee / kein Bock |
| Conclusion | Abschließend sei... | Zusammenfassend... | Also, kurz gesagt... |

### Formal ↔ Informal Equivalents

| Formell | Informell |
|---------|-----------|
| Ich möchte Sie darüber informieren, dass... | Ich wollte dir sagen, dass... |
| Es wäre wünschenswert, wenn... | Es wäre cool, wenn... |
| Ich bin der Auffassung, dass... | Ich finde, dass... |
| Bezug nehmend auf Ihr Schreiben... | Wegen deiner Nachricht... |
| Ich ersuche Sie, ... zu... | Könntest du vielleicht...? |
| Hochachtungsvoll | Liebe Grüße / Bis bald |
| Man sollte in Betracht ziehen, dass... | Man sollte mal überlegen, ob... |
| Angesichts der Tatsache, dass... | Weil halt... |

### Modalpartikeln (Informal Register Markers)

These small words add **nuance** in informal German:

| Partikel | Nuance | Example |
|----------|--------|---------|
| doch | emphasis/surprise | Das war **doch** klar! |
| mal | softening request | Kannst du **mal** helfen? |
| ja | shared knowledge | Das ist **ja** bekannt. |
| halt/eben | resignation | Das ist **halt** so. |
| eigentlich | actually | **Eigentlich** wollte ich... |
| schon | reassurance | Das wird **schon** klappen. |
| wohl | assumption | Er ist **wohl** krank. |

## 📝 Wortschatz (15 min) — Redemittel für alle Textsorten (All Text Types)

### Erörterung (Argumentative Essay) — FORMAL

| Phase | Redemittel |
|-------|-----------|
| Einleitung | Das Thema „..." wird in der Öffentlichkeit kontrovers diskutiert. |
| Pro | Dafür spricht, dass... / Ein wesentlicher Vorteil ist, dass... |
| Contra | Dagegen spricht, dass... / Kritiker weisen darauf hin, dass... |
| Eigene Meinung | Meines Erachtens... / Ich bin der Überzeugung, dass... |
| Schluss | Abschließend lässt sich sagen, dass... |

### Beschwerde (Complaint) — FORMAL

| Phase | Redemittel |
|-------|-----------|
| Betreff | Beschwerde über... / Reklamation betreffend... |
| Problem | Ich möchte mich über ... beschweren. / Leider muss ich feststellen, dass... |
| Forderung | Ich fordere Sie auf, ... / Ich erwarte, dass... |
| Frist | Ich bitte um Rückmeldung bis zum... |
| Schluss | Mit freundlichen Grüßen |

### Bericht (Report) — STANDARD/FORMAL

| Phase | Redemittel |
|-------|-----------|
| Einleitung | Der folgende Bericht befasst sich mit... |
| Darstellung | Wie aus den Daten hervorgeht, ... / Es zeigt sich, dass... |
| Bewertung | Positiv/Negativ zu bewerten ist, dass... |
| Empfehlung | Es wird empfohlen, ... zu... |

### Persönliche E-Mail — INFORMAL

| Phase | Redemittel |
|-------|-----------|
| Anrede | Liebe/r ..., / Hey ..., |
| Einleitung | Wie geht's dir? / Lange nichts gehört! |
| Erzählung | Stell dir vor, ... / Du glaubst nicht, was passiert ist! |
| Schluss | Meld dich mal! / Schreib bald zurück! / Bis bald! |

## 📚 Lesen (15 min) — Register Comparison

### Text A (Formal — Beschwerde)

> Sehr geehrte Damen und Herren,
>
> hiermit möchte ich mich über den Sprachkurs „Deutsch B2" (Kursnummer 4587) beschweren, den ich seit dem 1. September besuche. Leider muss ich feststellen, dass der Kurs nicht den in der Beschreibung angegebenen Standards entspricht. Die Unterrichtsmaterialien sind veraltet und die versprochenen Online-Ressourcen stehen nicht zur Verfügung.
>
> Ich fordere Sie auf, die Materialien zu aktualisieren oder mir die Kursgebühren anteilig zu erstatten. Ich erwarte eine Stellungnahme bis zum 30. Oktober.
>
> Mit freundlichen Grüßen,
> Maria Schmidt

### Text B (Informal — E-Mail an Freundin)

> Hey Lisa!
>
> Ich muss dir unbedingt erzählen, was in meinem Deutschkurs los ist. Also, der Kurs ist echt eine Katastrophe! Die Bücher sind mega alt und die Online-Sachen, die sie uns versprochen haben, gibt's einfach nicht. Hab mich jetzt beschwert, mal sehen, ob sich was tut. Bin echt genervt!
>
> Wie läuft's bei dir so? Meld dich mal!
> LG, Maria

### Analyse-Aufgaben

1. Listen Sie alle **formellen Merkmale** in Text A auf.
2. Listen Sie alle **informellen Merkmale** in Text B auf.
3. Welche **Nomen-Verb-Verbindungen** finden Sie in Text A?
4. Welche **Modalpartikeln** finden Sie in Text B?
5. **Gleiche Information, anderer Stil** — Was genau ist inhaltlich identisch in beiden Texten?
