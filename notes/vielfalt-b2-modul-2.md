## Lektion 4: Familie — „In die Fußstapfen der Eltern treten?"

### Grammatik

#### 1. Nomen-Verb-Verbindungen (Funktionsverbgefüge)
These are fixed noun-verb combinations where the verb has little meaning on its own. They are very common in formal/written German.

| Nomen-Verb-Verbindung | Einfaches Verb | Beispiel |
|------------------------|---------------|----------|
| eine Entscheidung **treffen** | entscheiden | *Die Eltern haben eine wichtige Entscheidung **getroffen**.* |
| eine Rolle **spielen** | wichtig sein | *Geld **spielt** dabei keine große **Rolle**.* |
| zur Verfügung **stehen** | verfügbar sein | *Das Stipendium **steht** Ihnen zur **Verfügung**.* |
| in Frage **kommen** | möglich sein | *Das **kommt** für mich nicht in **Frage**.* |
| Kritik **üben** (an +Dat) | kritisieren | *Sie **übt** Kritik **an** der Erziehung.* |
| Einfluss **nehmen** (auf +Akk) | beeinflussen | *Die Eltern **nehmen** Einfluss **auf** die Berufswahl.* |
| in Betracht **ziehen** | berücksichtigen | *Man sollte alle Optionen in Betracht **ziehen**.* |
| Rücksicht **nehmen** (auf +Akk) | berücksichtigen | *Er **nimmt** Rücksicht **auf** die Wünsche seiner Kinder.* |
| unter Druck **setzen** | drängen | *Die Eltern **setzen** ihre Kinder unter **Druck**.* |
| einen Kompromiss **schließen** | sich einigen | *Die Familie hat einen **Kompromiss** geschlossen.* |

#### 2. Konditionale Zusammenhänge: Bedingungen ausdrücken

| Ausdruck | Beispiel |
|----------|----------|
| wenn | *Wenn die Eltern zustimmen, studiert sie Kunst.* |
| falls | *Falls es nicht klappt, macht er eine Ausbildung.* |
| sofern | *Sofern die Noten stimmen, bekommt sie den Platz.* |
| vorausgesetzt, (dass) | *Vorausgesetzt, dass er die Prüfung besteht, kann er anfangen.* |
| unter der Voraussetzung, dass | *Unter der Voraussetzung, dass alles klappt, zieht sie um.* |
| es sei denn | *Er studiert Medizin, es sei denn, er bekommt keinen Studienplatz.* |
| andernfalls / sonst | *Er muss sich beeilen, andernfalls verpasst er die Frist.* |

> **Konjunktiv II for unreal conditions:**
> *Wenn ich mehr Zeit **hätte**, **würde** ich ein Praktikum machen.*
> *Hätte ich mehr Zeit, würde ich ein Praktikum machen.* (ohne *wenn*)

### Wortbildung: Nomen mit den Suffixen *-e*, *-nis* und *-ung*

| Suffix | Genus | Verb → Nomen | Beispiel |
|--------|-------|-------------|----------|
| **-e** | die | reisen → die Reise | *Die Reise hat ihn verändert.* |
| **-e** | die | fragen → die Frage | *Das ist eine schwierige Frage.* |
| **-nis** | das/die | erleben → das Erlebnis | *Das war ein unvergessliches Erlebnis.* |
| **-nis** | das/die | erkennen → die Erkenntnis | *Sie kam zu einer wichtigen Erkenntnis.* |
| **-ung** | die | entscheiden → die Entscheidung | *Die Entscheidung fiel ihm schwer.* |
| **-ung** | die | erziehen → die Erziehung | *Die Erziehung ist Aufgabe der ganzen Familie.* |
| **-ung** | die | erwarten → die Erwartung | *Die Erwartungen der Eltern sind hoch.* |

> **Regel:** Nomen mit *-ung* sind immer **feminin** (die). Nomen mit *-nis* sind meist **neutral** (das), manchmal **feminin** (die).

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Zustimmen | *Da stimme ich dir/Ihnen voll und ganz zu.* / *Das ist ein guter Punkt.* / *Genau, das sehe ich auch so.* |
| Widersprechen | *Da muss ich dir/Ihnen widersprechen.* / *Das sehe ich völlig anders.* / *Dem kann ich nicht zustimmen.* |
| Positiv bewerten | *Ich finde es gut/positiv, dass …* / *Ein Vorteil ist, dass …* / *Das hat den Vorteil, dass …* |
| Negativ bewerten | *Ich finde es problematisch, dass …* / *Ein Nachteil ist, dass …* / *Das hat den Nachteil, dass …* |
| Eine Diskussion moderieren | *Wollen wir mit … anfangen?* / *Was meinen die anderen dazu?* / *Können wir das zusammenfassen?* |
| Eine Meinung äußern | *Meiner Meinung nach …* / *Ich bin der Ansicht, dass …* / *Ich vertrete den Standpunkt, dass …* |
| Das Wort ergreifen | *Darf ich dazu etwas sagen?* / *Ich möchte da gern einhaken.* / *Entschuldigung, aber …* |

## Lektion 5: Soziale Medien — „Das eigene Profil schärfen"

### Grammatik

#### 1. Adverb-Adjektiv-Verbindungen
These combinations intensify or modify adjectives and are very common in written B2 German:

| Adverb | + Adjektiv | Bedeutung | Beispiel |
|--------|-----------|-----------|----------|
| überaus | erfolgreich | extremely successful | *ein **überaus** erfolgreicher Auftritt* |
| äußerst | wichtig | extremely important | *ein **äußerst** wichtiges Detail* |
| höchst | interessant | highly interesting | *ein **höchst** interessanter Beitrag* |
| wahnsinnig | beliebt | incredibly popular | *ein **wahnsinnig** beliebtes Profil* |
| ziemlich | gut | quite good | *ein **ziemlich** guter Tipp* |
| besonders | kreativ | especially creative | *ein **besonders** kreativer Ansatz* |
| unglaublich | schnell | unbelievably fast | *Das ging **unglaublich** schnell.* |

#### 2. Wortstellung: Ergänzungen und Angaben im Mittelfeld (vertieft)
Expanding on Lektion 1 — now with **Angaben** (adverbial info) mixed with **Ergänzungen** (complements):

**Full order:** Pronomen → temporal → kausal → modal → lokal → Akkusativergänzung

*Er hat **ihr** **gestern** **aus Freundlichkeit** **gern** **im Büro** **den Tipp** gegeben.*

> **Key point:** Definite information before indefinite: *Er hat **dem Chef** **einen Brief** geschrieben.* (not umgekehrt)

### Wortbildung: Zusammengesetzte Nomen
German loves compound nouns. The **last noun** determines gender and plural:

| Bestandteile | Kompositum | Genus | Englisch |
|-------------|------------|-------|----------|
| das Selbst + die Darstellung | die **Selbstdarstellung** | die | self-presentation |
| das Netz + das Werk | das **Netzwerk** | das | network |
| der Beruf + das Profil | das **Berufsprofil** | das | professional profile |
| das Video + das Tutorial | das **Videotutorial** | das | video tutorial |
| die Karriere + das Netzwerk | das **Karrierenetzwerk** | das | career network |
| die Selbst + das Marketing | das **Selbstmarketing** | das | self-marketing |

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Positiv bewerten | *Das macht einen guten Eindruck.* / *Das kommt bei den Leuten gut an.* |
| Negativ bewerten | *Das wirkt nicht sehr professionell.* / *Damit kann man sich schaden.* |
| Wirkung einer Person beschreiben | *Er/Sie wirkt … (kompetent/sympathisch/unsicher).* / *Er/Sie macht einen … Eindruck.* |
| Videotutorial einleiten | *Hallo und herzlich willkommen zu meinem Tutorial über …* / *Heute geht es um …* |
| Videotutorial beenden | *Ich hoffe, das Video hat euch gefallen.* / *Wenn ihr Fragen habt, schreibt sie in die Kommentare.* |
| Ratschläge geben | *Ich würde dir/Ihnen empfehlen, …* / *An deiner/Ihrer Stelle würde ich …* / *Du solltest unbedingt …* |

## Lektion 6: Teamarbeit — „Chefsache"

### Grammatik

#### Zweiteilige Konnektoren
These two-part connectors link contrasting, alternative, or additive ideas. Very important for B2 writing!

| Konnektor | Bedeutung | Beispiel |
|-----------|-----------|----------|
| sowohl … als auch | both … and | *Er ist **sowohl** kompetent **als auch** kreativ.* |
| nicht nur … sondern auch | not only … but also | *Sie arbeitet **nicht nur** hart, **sondern** ist **auch** sehr kreativ.* |
| entweder … oder | either … or | *Wir müssen **entweder** mehr Leute einstellen **oder** die Arbeitszeit verlängern.* |
| weder … noch | neither … nor | *Er hat **weder** Erfahrung **noch** die nötige Ausbildung.* |
| zwar … aber | admittedly … but | *Er ist **zwar** jung, **aber** sehr erfahren.* |
| einerseits … andererseits | on one hand … on the other | ***Einerseits** mag ich Teamarbeit, **andererseits** arbeite ich gern allein.* |
| je … desto/umso | the … the | ***Je** mehr Verantwortung er bekommt, **desto** motivierter arbeitet er.* |

> **Wortstellung bei *je … desto*:**
> *Je* + Komparativ → Nebensatz (Verb am Ende)
> *Desto* + Komparativ → Hauptsatz (Verb an Position 2, nach *desto + Komparativ*)
> ***Je** mehr man übt, **desto** besser **wird** man.*

### Wortbildung: Nomen mit dem Suffix *-(at)ion*

| Verb | Nomen (die …) | Beispiel |
|------|--------------|----------|
| organisieren | die Organisation | *Die Organisation des Projekts war gut.* |
| kommunizieren | die Kommunikation | *Gute Kommunikation ist das A und O.* |
| motivieren | die Motivation | *Seine Motivation ist sehr hoch.* |
| qualifizieren | die Qualifikation | *Sie hat die richtige Qualifikation.* |
| kooperieren | die Kooperation | *Die Kooperation mit dem Team klappt gut.* |
| präsentieren | die Präsentation | *Die Präsentation war überzeugend.* |

> **Regel:** Nomen mit *-(at)ion* sind immer **feminin** (die) und werden auf der letzten Silbe betont: Kommunika**ti**on.

### Kommunikation — Redemittel

| Funktion | Redemittel |
|----------|------------|
| Wichtigkeit ausdrücken | *Entscheidend ist, dass …* / *Darauf kommt es an.* / *Das spielt eine zentrale Rolle.* |
| Argumente nennen | *Ein wichtiges Argument ist, dass …* / *Dafür spricht, dass …* / *Man darf nicht vergessen, dass …* |
| Gegenargumente nennen | *Dagegen spricht, dass …* / *Ein Gegenargument wäre, dass …* / *Allerdings muss man bedenken, dass …* |
| Möglichkeiten nennen | *Eine Möglichkeit wäre, …* / *Man könnte auch …* / *Es besteht die Möglichkeit, dass …* |
| Sich rückversichern | *Habe ich das richtig verstanden?* / *Meinst du/Meinen Sie damit, dass …?* / *Wie genau ist das gemeint?* |
| Ein Fazit ziehen | *Zusammenfassend lässt sich sagen, dass …* / *Insgesamt kann man feststellen, dass …* / *Alles in allem …* |

## Extra Beruf: Vorstellungsgespräch

### Key phrases for job interviews

| Phase | Redemittel |
|-------|------------|
| Begrüßung | *Vielen Dank für die Einladung.* / *Ich freue mich, dass ich mich persönlich vorstellen darf.* |
| Über sich erzählen | *Ich bin seit … Jahren als … tätig.* / *In meiner letzten Position war ich verantwortlich für …* |
| Stärken nennen | *Zu meinen Stärken gehört …* / *Ich bin besonders gut darin, …* / *Meine Kolleg:innen schätzen an mir, dass …* |
| Motivation für die Stelle | *An dieser Stelle reizt mich besonders, dass …* / *Ich möchte mich in Richtung … weiterentwickeln.* |
| Fragen stellen | *Ich hätte noch eine Frage zu …* / *Mich würde interessieren, wie …* / *Könnten Sie mir mehr über … erzählen?* |
| Abschluss | *Vielen Dank für das Gespräch.* / *Ich freue mich auf Ihre Rückmeldung.* |

## Aussprache-Fokus Modul 2
1. **Die Konsonanten f, v und w** — v = [f] in native words (*Vater*), v = [v] in foreign words (*Visum*)
2. **Rhythmus in längeren Sätzen** — Keep natural stress and rhythm even in complex sentences
3. **Melodie und Betonung bei zweiteiligen Konnektoren** — Stress the contrasting/important element: *ZWAR jung, ABER erfahren*
